install git bash:
1. Clone project   
  Lệnh: git clone https://github.com/dungkhmt/CourseCPCBLS/

2. Copy source code + presentation của từng nhóm vào thư mục tương ứng, ví dụ CourseCPCBLS\OpenCBLS\src\khmtk60\miniprojects\G1\

3. Commit sản phẩm lên github
  Lệnh: git add CourseCPCBLS\OpenCBLS\src\khmtk60\miniprojects\G1\*
        git commit -m "add code group G1"
        git push origin master
